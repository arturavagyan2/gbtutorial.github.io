GitHub

What is Github?

GitHub is a website and cloud-based service that helps developers store and manage their code, as well as track and control changes to their code.

To understand exactly what GitHub is, you need to know two connected principles:

  • Version Control
  • Git

Version Control

Version control helps developers track and manage code changes to a software project. As a software project grows, version control becomes essential.

What is Git?

Git is a specific open-source version control system created by Linus Torvalds in 2005.

Git is a distributed version control system, meaning that the entire codebase and history are available on every developerâ€™s computer, allowing easy branching and merging.

Git Branching

What is Git Branching

At the start whe you create a repository it will be your master branch.So when you create a new branch it will have all the files that was in the master class but when you change anything in the new branch it will not affect the master branch.

When to use branching?

we will use it when we want to test a new feature but we dont want to it affect the main branch which is master if anything happens. other use is when we want to fix the main(master) brach.

Types of Branching

  • HotFix Branch
  • Feature Branching

How Branshes look like

Creating a new branch

  • If you want to check which branch you are in in the terminal write (git branch)
  • To create new branch (git checkout -b "name") name is the name of the beanch
  • To switch between branches (git chechout "name") name is the name of the beanch
  • To check the difference between your branches you can do (git diff "name") name is the name of the branch you want to compare it with the brach you are in.
  • To delete a branch (git branch -d "name")

How to Merge branches

  • All you have to do is write (git merge "name") name is the branch you want to merge to.
  • But sometimes it give you conflects.
  • if you have a conflect during merging check your code in the IDE or vscode and it wll show you why there was conflect and change it and try merging again or join everying which is not recommended.
